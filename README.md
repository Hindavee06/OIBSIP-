# OIBSIP- Task 2 Calculator
This task is a functional calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and is developed as part of the OIBSIP Web Development Internship.


## 🚀 Features

- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Clear (AC) and Delete functionality
- Responsive and user-friendly interface
- Button click interactions
- Real-time calculation display


##  Technologies Used

- HTML5 - Structure
- CSS3 - Styling
- JavaScript - Functionality
